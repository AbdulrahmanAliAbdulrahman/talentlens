# TalentLens — منصة اكتشاف المواهب (عرض لمعرض الاعتماد البرمجي)

منصة ويب عربية (RTL) تقوم بـ:
- اختبار قصير (12 سؤال)
- حساب درجات 8 مجالات مواهب
- رسم Radar + أفضل 3 مواهب
- توصيات تخصصات/وظائف + خطة 14 يوم
- تنزيل تقرير PDF بنقرة واحدة (يلتقط صفحة النتائج كما هي)

> **ملاحظة:** البيانات تُحفظ محليًا في المتصفح (LocalStorage) لتسهيل العرض بدون أي Backend.

---

## تشغيل المشروع

### 1) المتطلبات
- Node.js 18+ (يفضل 20+)
- npm أو pnpm

### 2) التثبيت والتشغيل
```bash
npm install
npm run dev
```

ثم افتح:
- http://localhost:3000

---

## صفحات المشروع
- `/` صفحة رئيسية (عرض + زر بدء)
- `/quiz` الاختبار
- `/results` تقرير النتيجة (مع تنزيل PDF)
- `/history` سجل النتائج + تصدير CSV
- `/dashboard` لوحة إحصائيات (Dashboard)

---

## كيف تبهر لجنة المعرض (سيناريو عرض 90 ثانية)
1) افتح الصفحة الرئيسية وقل: "هذه منصة تساعد الطالب يكتشف مواهبه خلال 3 دقائق".
2) ادخل اسم، وأجب بسرعة على الأسئلة (أو جهّز نتيجة مسبقًا في السجل).
3) عند النتيجة:
   - اشرح "أفضل 3 مواهب" + لماذا (ملخص التحليل)
   - ركّز على الرسم Radar لأنه بصري ويقنع بسرعة
4) اضغط "تنزيل PDF" وقل: "يمكن طباعة التقرير وتسليمه للمرشد الأكاديمي".
5) افتح صفحة السجل وقل: "النظام يحفظ النتائج ويتيح تصديرها CSV".
